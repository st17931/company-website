About the Course

This course is designed for those interested in mastering software development in the fields of web, mobile app, and desktop solutions. We provide a comprehensive curriculum to ensure that you acquire the skills needed to build modern applications.

  • Duration: 3 Months
  • Mode: Online (Flexible hours)
  • Prerequisites: Basic programming knowledge
  • Level: Beginner to Intermediate

Course Syllabus

Week 1: Introduction to .NET

  • History and evolution of .NET Framework and .NET Core
  • Overview of .NET 5/6/7 and .NET Runtime
  • Installation of Visual Studio and Visual Studio Code
  • Understanding CLR, CTS, and CLS
  • First C# program: Hello World

Week 2: C# Basics

  • Variables, data types, and type conversion
  • Control structures (if, switch, loops)
  • Arrays and collections
  • Input/Output handling
  • Debugging basics in Visual Studio

Week 3: Object-Oriented Programming (OOP)

  • Classes, objects, and methods
  • Access modifiers and encapsulation
  • Inheritance and polymorphism
  • Abstract classes and interfaces

Week 4: Advanced C#

  • Delegates and events
  • Generics
  • Exception handling
  • Working with DateTime and File I/O

Week 5: Introduction to Databases

  • Relational database concepts
  • SQL basics (SELECT, INSERT, UPDATE, DELETE)
  • Working with SQL Server
  • Introduction to database design (ER diagrams)

Week 6: Entity Framework Core Basics

  • Setting up Entity Framework Core
  • Code-first and database-first approaches
  • Creating models and DbContext
  • CRUD operations using EF Core
